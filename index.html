Frontend Shortcuts

1. HTML

  • <h1> to <h6>: Headings
  • <div>: Division element
  • <a>: Link
  • <img>: Image

2. CSS

  • px: Pixel unit
  • em/rem: Relative units
  • @media: Media queries for responsive design
  • z-index: Element stacking order

3. JavaScript

  • DOM: Document Object Model for interacting with page elements
  • ES6: Modern JavaScript features like let and const
  • API: Interfaces for system integration

4. Libraries and Frameworks

  • React: Library for building user interfaces
  • jQuery: Simplifies DOM manipulation
  • Lodash: Utility functions for data processing
  • Bootstrap: Responsive web design framework
  • Angular: Framework for building powerful web applications
